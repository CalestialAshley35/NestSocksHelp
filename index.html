Documentation

Purpose: The NestSocks Theme Selector allows users to easily choose a theme or animation style from the NestSocks framework and provides direct links for integration into their projects.

Features:

  • Dropdown menu for selecting themes and animations.
  • Dynamically updates and displays the required CSS or JavaScript links for integration.
  • Special support for Star Animation, providing both the CSS and JavaScript links.

How to Use:

  1. Include this file in your project and open it in a browser.
  2. Select a theme or animation from the dropdown menu.
  3. Copy the displayed links and use them in your project.
